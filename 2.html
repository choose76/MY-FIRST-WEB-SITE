CSS

CSS는 캐스케이딩 스타일시트(Cascading Stylesheet)의 줄임말입니다. 여기에서 캐스케이딩(Cascading)이란 ‘폭포같은, 계속되는, 연속적인’이라는 의미를 가진 단어인데, 이는 곧 ‘위에서부터 아래로 내려가며 이어진다’는 의미이기도 합니다. 실제로 CSS 코드가 HTML 문서에 스타일을 입히는 방식은 캐스케이딩이라는 단어가 가진 의미와 유사합니다. HTML 코드는 HTML이 제공하는 태그의 모음입니다. 그리고 코드의 구조는 하나의 태그가 다른 태그를 포함하거나, 또 다른 태그에 포함되는 형태의 ‘계단식 구조’를 지니고 있습니다.
